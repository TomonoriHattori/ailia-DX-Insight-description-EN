Translate Document Files

Translate Document Files

By registering document files to an index, you can translate registered document files using the AI function.
You can also directly enter the document into the chat box at the bottom center to translate that document.

Look here for how to register files to indexes.

  1. Select "Text Translation" from the app menu at the top left of the screen.

  2. Right-click the file you want to translate and click "Translation". The supported file formats are pdf, txt, md, docx, xlsx.

  3. The AI-generated translation result will be displayed. You can save the translated result as a file by clicking the “Save” button at the bottom right.

  4. When you translate a PDF file, you can output it as a PDF.(*1)


Translating Text

Text written in Japanese will be translated into English, and text written in other languages will be translated into Japanese.

  1. Select "Text Translation" from the app menu at the top left of the screen.

  2. Enter the text you want to translate into the chatbox at the bottom center and send it.
  3. The AI-generated translation result will be displayed.

(*1)Additionally, PDFs have a very complex structure, and ailia DX Insight estimates the layout internally. Depending on the type of PDF, the layout may be disrupted after translation.
